# Mobile-Sales-Dashbord
Power BI dashboard project designed to analyze and visualize mobile sales data in an interactive and user-friendly way. This dashboard helps track sales performance, revenue trends, customer insights, and product-wise analysis.

Features
Interactive Power BI dashboard
Sales performance analysis
Brand-wise and product-wise sales tracking
Monthly and yearly revenue trends
Customer and regional insights
Data visualization with charts and KPIs
Easy-to-understand business insights
